# AI-Powered Skin Cancer Detection System
## Overview
Healthcare AI project using deep learning and neuroevolution to classify skin lesions.

## Technologies
- Python
- TensorFlow/Keras
- ResNet50
- NEAT-Python
- Scikit-learn

## Dataset
ISIC skin lesion dataset.

## Features
- Image preprocessing
- Transfer learning
- NEAT optimisation
- Performance evaluation
- Confusion matrix analysis

## Results
Accuracy: 62%
Precision: 73%
Recall: 37%
F1 Score: 49%

## Future Improvements
- Grad-CAM explainability
- Improved malignant recall
- Streamlit deployment
